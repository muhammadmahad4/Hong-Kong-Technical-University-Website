Use of const

const is basically another way to declare a variable. However, the variable declared by const will have 2 properties. It will be block scoped(like let) and its value cannot be changed. So thats why I used to declare varibles which did not require to be changed. for example, I used const to declare the msgs variable which was an array containing all the msgs to show on program page.

Use Of Positioning:


I have used position in three places:

  1. In homepage: I used relative positioning for click to apply now image. I did Css code for this on line 64-66 of HomePage.css. First, I selected the "click to apply now" image using class selector and then I set its position to relative (position:relative) and then I gave left property value of 14% (left:14%) to give 14% space from left side relative to the initial position.
  2. I used absolute positioning on visit page to position the "check availability" and "reset" buttons. Firstly, I set the parent container (form element) to position:relative on line 77 to 79 in visit.css, so that parent is positioned. Then, I selected both the buttons parent using .button class (class selector) and then applied absolute positioning and gave left property a value of 12% on lines 80 to 83 to position it 12% from left relative to the form element (parent).
  3. I also used positioning on Apply page to ensure that when cursor hovers over major application form, headings background color is still white. On line 9, I set the “.Apply-block” position to relative to make the headings parent positioned. Then I set the position of all the headings to absolute on line 31, and then I gave top/left values to each heading by selecting them using id selector from lines 36-50.
